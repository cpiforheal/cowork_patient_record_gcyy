# 内测交付与操作文档索引

本目录用于第一轮内测和交付说明，面向一线岗位、科室负责人、物资管理员、领导和负责部署维护的人员。这里的文档不要求阅读代码，也不要求理解前后端架构。

## 谁应该看哪一份

| 阅读对象 | 建议先看 | 用途 |
| --- | --- | --- |
| 院内领导、科室负责人 | [项目介绍与内测边界](project-overview.md)、[领导检查与验收指南](leader-check-guide.md) | 快速理解系统解决什么问题、每天看什么、怎样判断内测是否有效 |
| 前台、医生、护士、检验/检查岗位 | [岗位操作说明](role-operation-guide.md) | 按岗位了解每天要点哪里、填什么、怎样闭环 |
| 科室申请人、科室负责人、库管 | [岗位操作说明](role-operation-guide.md)、[进销存操作指南](../inventory-operation-guide.md) | 完成物资申领、审核、发放、签收、周消耗、盘点和追溯 |
| 系统管理员、信息科、交付人员 | [环境适配与部署交接](environment-and-deployment.md)、[内测复核与验收清单](pilot-review-and-acceptance.md) | 准备主机、配置数据库和附件目录、启动系统、验收和备份 |

## 文档定位

- 本目录是“交付说明层”，用于培训、上线前说明和内测验收。
- 已有文档仍然保留：部署细节见 [内测部署检查清单](../pilot-deployment-checklist.md)，交付包使用见 [内测交付包说明](../pilot-package-readme.md)，进销存细节见 [进销存操作指南](../inventory-operation-guide.md)。
- 本轮固定系统原型，不再调整前后端交互、安全策略和数据权限校验；内测阶段重点放在岗位培训、操作闭环、数据备份和问题记录。
