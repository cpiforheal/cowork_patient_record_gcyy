# 岗位操作说明

本文面向日常使用人员。每个岗位只需要先掌握自己相关的入口，不需要一开始理解全部菜单。

## 通用登录流程

1. 打开系统地址，例如 `http://内测主机IP:8080/`。
2. 选择所在科室。
3. 选择自己的账号。
4. 输入密码并登录。
5. 登录后优先进入自己岗位相关的工作台或待办入口。

固定终端建议使用同一浏览器访问系统，不要频繁清理浏览器数据，以免每次都重新选择登录信息。

## 前台或登记岗位

每天主要做三件事：

1. 新建或查找患者。
2. 补齐患者基础信息。
3. 确认患者进入正确流程，便于医生、护士或检查岗位继续处理。

注意事项：

- 患者姓名、联系方式、就诊日期等基础信息尽量一次录准。
- 不确定的信息先留空或备注，不要随意填猜测值。
- 发现重复患者时，先核对身份证、手机号、就诊时间等关键字段，再决定是否继续新建。

## 医生、护士、检验和检查岗位

每天主要做四件事：

1. 从今日患者或患者流程进入患者详情。
2. 填写本岗位负责的病历字段。
3. 上传必要附件，例如图片、PDF 或检查资料。
4. 保存后确认页面没有未保存提示。

建议习惯：

- 先填必须字段，再补充说明字段。
- 上传附件后打开预览确认能查看。
- 遇到保存失败时，不要立刻关闭页面，先截图并记录患者姓名、时间和操作步骤。
- 长表单填写时按章节逐段完成，减少漏填。

## 质控或管理员岗位

每天主要检查：

- 是否存在待质控记录。
- 是否存在附件作废、恢复或异常上传记录。
- 患者流程是否有长期停留在某一步的情况。
- 操作记录中是否有需要复核的异常动作。

处理原则：

- 先处理会影响患者资料完整性的事项。
- 对作废、恢复、异常修改等动作保留原因说明。
- 需要一线补填时，明确指出患者、字段和原因，避免笼统退回。

## 科室物资申请人

入口：`进销存管理 -> 进销存主控台` 或 `申领审批`。

需要物资时：

1. 选择“提交申领”。
2. 选择物资、规格和数量。
3. 填写用途、申请人、负责人和预计使用周。
4. 提交后等待负责人审核。
5. 物资发放后及时签收。

注意事项：

- 申领数量尽量按实际需要填写，不要把“可能会用到”全部一次性填入。
- 领取后必须签收，否则系统里会一直显示未闭环。
- 本周实际消耗和下周预计领用应按固定时间填写，减少口头统计。

## 科室负责人

入口：`进销存管理 -> 申领审批`。

每天主要做：

1. 查看待审核申领。
2. 核对申请理由、数量和近期消耗是否合理。
3. 审核通过后等待库管发放。
4. 关注本科室长期未签收、异常消耗和盘点差异。

审核重点：

- 是否确实为本科室工作需要。
- 数量是否明显高于近期使用量。
- 是否存在重复申领或用途不清。
- 高值、敏感、临期物资是否需要额外说明。

## 物资管理员或库管

入口：`进销存管理` 下的 `库存看板`、`物资档案`、`申领审批`、`盘点处置`、`追溯流水`。

每天主要做：

1. 查看低库存、临期、过期和待发放事项。
2. 新物资先建档，再做入库。
3. 对已审核申领进行发放。
4. 处理退回、报废和盘点差异。
5. 必要时导出库存或流水，留作台账。

操作习惯：

- 入库前确认物资名称、规格、单位统一。
- 有批号、效期要求的物资必须认真填写批号和效期。
- 发放时核对数量和批次，避免线下发了但系统未扣减。
- 报废、盘亏、退回必须填写原因，便于后续解释。

## 领导或管理查看岗位

入口：`进销存管理 -> 领导驾驶舱`、`进销存主控台`、`追溯流水`。

每天优先看：

- 今日是否有紧急风险。
- 低库存、临期、过期、报废和盘点差异是否有人处理。
- 申领是否长期停在待审核、待发放或待签收。
- 科室消耗是否明显异常。

每周建议看：

- 周消耗是否填报完整。
- 哪些科室申领频率或数量明显偏高。
- 哪些物资库存过低或长期不动。
- 导出关键数据留档，形成内部检查依据。

## 遇到问题时怎么记录

为了便于复核，请尽量记录下面信息：

- 操作人和科室。
- 患者姓名或物资名称。
- 发生时间。
- 当前页面和按钮名称。
- 系统提示文字。
- 截图或简短录屏。

如果问题涉及附件打不开、数据保存失败、库存数量不对，先不要重复多次提交，记录现场后交由管理员核查。
